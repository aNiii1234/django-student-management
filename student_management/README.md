# 学生信息管理系统

一个功能完整的Django学生信息管理系统，实现用户注册、登录、权限分离和完整的数据管理功能。

## 系统功能

### ✅ 已实现功能

#### 1. 用户认证系统
- **用户注册**：完整注册表单，包括基本信息、角色选择
- **用户登录**：安全的登录系统，支持不同角色重定向
- **权限分离**：管理员、普通用户角色分离
- **个人资料管理**：用户可查看和编辑个人资料

#### 2. 数据库设计（5张核心表）
1. **用户表 (User)**：扩展Django用户模型，增加角色、手机号等字段
2. **学生档案表 (StudentProfile)**：学生详细信息
3. **院系表 (Department)**：院系信息管理
4. **专业表 (Major)**：专业信息管理
5. **课程表 (Course)**：课程信息管理
6. **选课记录表 (Enrollment)**：学生选课和成绩管理

#### 3. 完整的CRUD操作
- 所有5张表都实现了完整的增删改查功能
- 管理员可以管理所有业务数据
- 学生用户可以查看和管理自己的数据

#### 4. 权限控制
- **管理员权限**：
  - 管理所有用户信息
  - 管理院系、专业、课程数据
  - 管理学生档案
  - 管理选课记录和成绩
  - 访问管理员面板

- **学生权限**：
  - 查看和编辑个人资料
  - 查看自己的选课记录
  - 查看个人成绩
  - 访问学生面板

## 快速开始

### 1. 环境准备
```bash
# 创建并激活虚拟环境
python -m venv venv
venv/Scripts/activate  # Windows

# 安装依赖
pip install django
```

### 2. 数据库迁移
```bash
cd student_management
python manage.py makemigrations
python manage.py migrate
```

### 3. 创建管理员账号
```bash
# 运行创建脚本
python create_admin.py

# 或使用命令
python manage.py createsuperuser
```

### 4. 启动开发服务器
```bash
python manage.py runserver
```

## 功能模块详解

### 1. 用户管理
- 注册新用户，可选择管理员、学生、教师角色
- 用户登录和权限验证
- 个人资料查看和编辑
- 管理员可查看所有用户

### 2. 学生档案管理
- 创建学生档案（自动创建对应用户）
- 编辑学生信息
- 删除学生档案
- 查看学生详细资料

### 3. 院系专业管理
- 院系信息的增删改查
- 专业信息的增删改查
- 院系与专业的关联管理

### 4. 课程管理
- 课程信息的增删改查
- 课程类型管理（必修、选修、实践）

### 5. 选课成绩管理
- 学生选课记录管理
- 成绩录入和修改
- 成绩等级评定
- 学生查看个人选课和成绩

- 学生面板：显示个人信息和选课情况

## 技术特性

- **框架**：Django 5.1.2
- **数据库**：SQLite（开发环境）
- **前端**：Bootstrap 5 + Font Awesome
- **权限控制**：基于角色的访问控制
- **国际化**：支持中文界面
- **响应式设计**：适配各种设备屏幕

## 安全特性

- 用户密码加密存储
- CSRF保护
- 基于角色的权限控制
- 安全的登录重定向
